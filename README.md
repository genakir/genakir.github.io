# Gennadii Kiryushatov — Resume Site

Static personal resume/business-card site designed to be deployed with GitHub Pages through GitHub Actions.

## Deploy with GitHub Actions

1. Create a new GitHub repository, for example `resume-site`
2. Upload these files to the root of the repository
3. Go to **Settings → Pages**
4. Under **Build and deployment**, set **Source** to **GitHub Actions**
5. Push to `main`, or run **Actions → Deploy resume site to GitHub Pages → Run workflow**

Your site URL will usually be:

```text
https://YOUR_GITHUB_USERNAME.github.io/resume-site/
```

For a user site repository named exactly `YOUR_GITHUB_USERNAME.github.io`, the URL will usually be:

```text
https://YOUR_GITHUB_USERNAME.github.io/
```
